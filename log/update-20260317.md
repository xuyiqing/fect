## What Changed

Four workstreams: CFE convergence conditioning, book-derived tests, plot refactor (pre/post colors + esplot harmonization), and Quarto book restructure with content updates.

### 1. CFE Convergence Conditioning (`7b33c98`)

Applied component-wise convergence pattern from `ife_sub.cpp` to `cfe_sub.cpp`: `max(dif, dif_inter)` for interactive FE when `r > 0`, `+1e-10` denominator safety on all convergence ratios, stop-burnin transition. Zero speed regression confirmed.

### 2. Book-Derived Tests (`7418aa0`)

98 new tests in `test-book-claims.R` covering documented claims from the Quarto book: method equivalences (gsynth ≡ ife+nevertreated, IFE(r=0) ≡ FE), force parameter variations, diagnostic tests, output structure, edge cases, MC method, CFE features, inference methods, plot types, CV criterion selection, bundled dataset properties.

### 3. Plot Refactor

**Pre/post color distinction** in gap plots:

- Unconnected mode: pre-treatment estimates in gray (`pre.color = "gray50"`), post-treatment in black (`post.color` inherits `color`)
- Connected mode: pre-treatment long-dashed line + muted ribbon, post-treatment solid line + ribbon
- New params `pre.color` and `post.color` in both `esplot()` and `plot.fect()`
- Users can customize: `plot(out, pre.color = "steelblue", post.color = "firebrick")`

**esplot() harmonization with plot.fect()**:

- `esplot()` now accepts fect objects directly: `esplot(out)` produces same plot as `plot(out, type = "gap")`
- Harmonized defaults: `show.points = TRUE` (was FALSE), `show.count = TRUE` when count data available
- `cex.*` scale unified: esplot now accepts pt values directly (16, 15, 15, 5), no more divide-multiply roundtrip
- `count.color` spelling harmonized to `"gray70"` (was `"grey70"` in plot.fect)
- Extracted `.add_connected_layers()` helper in esplot.R to reduce duplication

**Core cap**: Default parallel cores raised from 4 to 8 in `default.R` and `fect_nevertreated.R`. Boxed message shows core usage at runtime.

25 new tests in `test-plot-refactor.R`. 590/590 total tests pass.
